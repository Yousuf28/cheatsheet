put in .vimrc file  
```
set relativenumber
set numberwidth=2
set cpoptions+=n
inoremap kj <esc>
vnoremap kj <esc>
syntax enable
colorscheme monokai
```

```mkdir ~/.vim/colors```  
if directory not exist. then  
 [download monokai](https://github.com/sickill/vim-monokai)    
 and move to colors  directory.
